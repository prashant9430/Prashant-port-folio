# Prashant Kumar Pandey — Portfolio Website 🌌

> A cinematic, interactive developer portfolio built with cutting-edge web technologies.



![Portfolio Preview](public/images/preview.png)



---

## ✨ Features

- 🎬 **GSAP Animations** — Smooth, cinematic transitions and scroll-triggered effects
- 🌐 **Three.js / WebGL** — 3D visuals and immersive canvas experiences
- ⚡ **React + TypeScript** — Scalable, type-safe component architecture
- 🎨 **Custom Shaders** — WebGL-powered visual effects
- 📱 **Responsive Design** — Optimized across all screen sizes

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React + TypeScript | UI & Component Logic |
| GSAP | Animations & Scroll Effects |
| Three.js | 3D Rendering |
| WebGL | Custom Shader Effects |
| HTML / CSS / JS | Structure & Styling |

---

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/prashant9430/Prashant-port-folio.git

# Navigate into the project
cd Prashant-port-folio

# Install dependencies
npm install

# Start development server
npm run dev
⚠️ Important Note on GSAP Plugins
This repo uses GSAP Trial Plugins (free version) as a substitute for Club plugins.
🔴 Trial plugins cannot be used for hosted/production deployments.
To use Club plugins (required for hosting), get them here:
```
